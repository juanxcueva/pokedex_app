# Pokedex_app

App construida en Flutter utilizada para mostrar datos acerca de los pokemons.

## Acerca del proyecto

El proyecto esta construido con las bases de clean architecture, presenta una pantalla de splash que sirve como una pantalla que ve el usuario mientras se cargan los datos de la API, un home_page que contiene todos los pokemons en cards y la pagina de details_page que contiene detalles acerca de cada pokemon tales como su nombre,tama√±o, peso, evolucion,etc.
